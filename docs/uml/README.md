# Diagramas UML - Modulo de autenticacion y acceso por tenant

Este directorio contiene los diagramas UML base del modulo documentado:

- `caso-uso.md`: interacciones principales del usuario con el modulo.
- `clases.md`: clases y componentes principales relacionados con autenticacion, tenant y JWT.
- `secuencia-login.md`: flujo de inicio de sesion con validacion de tenant y generacion de token.

Los diagramas estan escritos en Mermaid para que puedan visualizarse directamente en GitHub o exportarse como imagen para el documento Word de entrega.
